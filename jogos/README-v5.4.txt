RealMind — Jogos v5.4 (Mind & Focus)

Como instalar no GitHub Pages (raiz do projeto):
1) Faça upload da pasta `jogos/` inteira para o repositório (na raiz do RealMind).
2) Dentro de `jogos/`, crie a subpasta `sons/` e adicione os arquivos MP3 abaixo:

Necessários:
- sons/pop.mp3         (bolhas)
- sons/vento.mp3       (respiração – som de vento leve)
- sons/chuva.mp3       (chuva relaxante em loop)
- sons/trovao.mp3      (relâmpago curto)
- sons/ding.mp3        (acertos/feedbacks)
- sons/beep.mp3        (erro/feedbacks)
- sons/relax.mp3       (ambiente para foco auditivo)
- sons/simon1.mp3
- sons/simon2.mp3
- sons/simon3.mp3
- sons/simon4.mp3

Após subir os arquivos, acesse:
- /jogos/jogos.html (menu)
- ou cada jogo diretamente (ex.: /jogos/jogo-areia.html)

Observações:
- Os sons só tocam após interação do usuário (clique/toque).
- Todos os jogos funcionam offline (PWA) após o primeiro carregamento.
- Não há salvamento de progresso.
